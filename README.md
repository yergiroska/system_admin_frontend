# System Admin Frontend

Frontend React para el sistema de administración y análisis de datos.

## Stack

- **Frontend**: React, Vite
- **Routing**: React Router v6
- **Gráficas**: Recharts
- **HTTP**: Axios
- **Auth**: JWT

## Módulos

- Login con autenticación JWT
- Dashboard con gráficas en tiempo real
- Empresas con búsqueda y paginación
- Productos con búsqueda y paginación
- Clientes con segmentación ML (VIP, Medio, Dormido)
- Compras con búsqueda y paginación

## Instalación
```bash
npm install
npm run dev
```

## Requisitos

Tener corriendo el backend `system_admin_api` en `http://localhost:8000`
